# RealBoard - Client #
Realtime collaborative IDE and simple project management

## Feature list :
* Realtime collaborative IDE *(Skip all conflict in writing)*
* Simple Project Management *(Create new / Get report)*
* Simple Task Manager
* Live discussion
* Live Notification
* Firebase Integration
